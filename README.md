# Project Template

项目模板，用于快速搭建项目框架。

## 模板类型

- [x] Node_Express
- [x] Node_Koa
- [x] React
- [x] React_Ts
- [x] Uniapp
- [x] Uniapp_Ts
- [x] Uniappx_Uts
- [x] Vue2
- [x] Vue3
- [x] Vue3_Ts
- [x] Wechat

## 使用方法

```bash
# 安装项目模板
npm install -g @Cicada-coding/project-template

# 创建项目
Cicada create <project-name>
```
